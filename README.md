## English Description

### One-line Pitch
A local-first AI knowledge base powered by RAG - Upload documents, get intelligent answers.

### Detailed Description
Wali AI Knowledge Base is a cross-platform desktop application built with Tauri + Vue 3 + Rust. It implements complete RAG (Retrieval-Augmented Generation) functionality by integrating Alibaba Cloud's Qwen API. After uploading documents, the system automatically chunks, vectorizes and stores them. When users ask questions, AI provides accurate answers based on document content with source citations. All data is stored locally for privacy protection. Supports TXT, Markdown, PDF and more, processing 1MB documents in just 10 seconds.
